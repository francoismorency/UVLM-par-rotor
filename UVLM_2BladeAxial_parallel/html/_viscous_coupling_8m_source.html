4 % ViscousCoupling.m reads data that is the result of a UVLM calculation
5 % and uses the data to compute the Effective angle of attack and viscous
6 % lift on the blade sections. The UVLM calculation must be done first and
7 % its exported results should be available in the same folder as
8 % ViscousCoupling.m. The csv files from UVLM should be also grouped
9 % together in an excel file named Data.xlsx to facilitate reading the data.
10 % When running ViscousCoupling.m, attention must be made for points of
11 % singularities, especially found near the root of the blade, when flow
12 % separation occurs or at the first few rotor revolutions. In the case when
13 % the code sees a singularity, it will fail. The user has to revise the
14 % inviscid lift data and eliminate them from the analysis.

132 function [Coefficient]=GetAlfa(A,B,Alfa)

136 if Alfa >= A(i) && Alfa <= A(i+1)
137 Interpolation=(Alfa-A(i))/(A(i+1)-A(i));
